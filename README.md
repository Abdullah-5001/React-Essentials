# React Essentials Starter

A small React + Vite practice project for learning component basics, props, and state-driven UI.

## Setup

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

- `src/App.jsx` - main application layout and tab state
- `src/components/` - reusable UI components
- `src/data.js` - content data used by the app
- `src/index.css` - global styles

## Notes

This project uses React 19 and Vite.
